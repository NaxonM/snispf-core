# SNISPF Documentation

This directory contains all project documentation except the repository root `README.md`.

## Documents

- `beginner-guide.md`: first-time setup, safe defaults, and troubleshooting order.
- `api-contract.md`: control service API schema and compatibility notes.
- `internals.md`: runtime architecture and strategy internals.
- `examples.md`: how to use the bundled sanitized config examples.
- `roadmap.md`: planned evolution and non-goals.

## Reading order

1. `beginner-guide.md`
2. root `README.md`
3. `api-contract.md` and `internals.md` when integrating or debugging
4. `roadmap.md` for planned direction
